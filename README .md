# Subaru Weekly Report

Interactive weekly task report for Subaru SA — dev, web maintenance, and QA tracking, built for Algorithm.

## Sections
- **Overview** — week summary and status snapshot
- **Weekly Breakdown** — completed, in progress, and scheduled tasks
- **Task Details** — form testing, plugin audit, and API check breakdowns
- **Issues & Fixes** — QA issues found and resolutions

## Usage
Open `subaru-weekly-report.html` directly in a browser, or view the hosted version via GitHub Pages (see below).
