Section 04 · Quality assurance

Issues found & fixes

!

Book a Test Drive — WRX: notification email misrouted

Resolved
What we found

The notification rule was set to send to info@subaru.co.za when Vehicle Model = "Subaru Bellville." Bellville is a dealer, not a vehicle model, so the condition never matched correctly and WordPress rejected the notification with "Cannot send email because the TO address is invalid."

Root cause

Notification routing was built on the wrong form field. It should have been keyed off Select Dealer, not Vehicle Model.

Fix applied

Rebuilt the notification condition to trigger off the Select Dealer field instead, so Bellville leads (and every other dealer) route to the correct inbox. Confirmed fixed with a live test submission — webhook fired and the lead synced successfully to the CRM.
